# Standalone Kotlin File

## Run Process
- Download Maven Dependencies
- Compile the Kotlin file using `kotlinc`
- Run a one-liner Java execution

## Editor - Just open IntelliJ
- Download Maven Dependencies
- Create `.iml` file
- Launch IntelliJ with IML file
